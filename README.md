# Excel 测试工具集

本仓库包含了一系列用于自动化测试的Excel工具。

## 主要功能

- DY循环测试
- 数据验证
- 自动化报告生成

## 文件说明

### 测试工具

- `测试工具Ver_4.0.1.xlsm`
  - 主要测试工具文件
  - 包含VBA自动化脚本
  - 支持批量数据处理

- `DY循环-DY.xlsx`
  - DY循环测试数据文件
  - 用于存储测试结果

## 使用说明

1. 打开 `测试工具Ver_4.0.1.xlsm`
2. 启用宏
3. 按照界面提示进行操作

## 注意事项

- 使用前请确保Excel安全设置允许运行宏
- 建议定期备份数据
- 如遇问题请查看日志文件
